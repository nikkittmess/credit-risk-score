##Credit Risk Analysis using XGBoost & SMOTE:
Built a machine learning model to predict loan approval and classify credit risk levels. Applied SMOTE to handle class imbalance and used XGBoost for high-performance classification. Performed feature engineering and evaluated the model using standard classification metrics.
